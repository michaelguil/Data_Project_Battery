# [팀 프로젝트] 2차 전지 화성 공정 최적화를 통한 불량률 개선 분석

## 1. 문제 정의 (Problem Definition)

2차 전지 제조의 마지막 단계인 화성 공정은 제품의 품질과 성능을 결정하는 핵심 공정입니다. 하지만 온도, 전압, 전류 등 제어해야 할 변수가 많고, 미세한 조건 변화에도 불량률이 크게 영향을 받습니다. 본 프로젝트에서는 약 3.13%에 달하는 공정 불량률을 데이터 기반으로 분석하여, 비용 손실을 최소화하고 생산성을 극대화하는 것을 목표로 삼았습니다.

## 2. 분석 목표 (Goal)

*   핵심 영향 변수(KPI) 식별: 공정 데이터 분석을 통해 불량률에 가장 큰 영향을 미치는 주요 공정 변수를 도출합니다.
*   불량 예측 모델 개발: 머신러닝을 활용하여 각 공정 단계별 셀(Cell)의 불량 여부를 예측하는 분류 모델을 구축합니다.
*   공정 최적화 시뮬레이션: 개발된 예측 모델을 기반으로, 핵심 공정 변수(온도)를 제어했을 때 불량률이 얼마나 개선되는지 시뮬레이션하고, 그 효과를 정량적으로 제시합니다.

## 3. 사용 기술 (Tech Stack)

*   언어: Python
*   라이브러리: Pandas, NumPy, Scikit-learn, CatBoost, LightGBM, XGBoost, Matplotlib, Seaborn

## 4. 분석 및 모델링 과정 (Analysis & Modeling Process)

### 4-1. 데이터 전처리 및 탐색적 데이터 분석 (EDA)

*   39,019개의 셀(Cell) 기준 공정 데이터(109개 변수)를 기반으로 변수 간 관계, 분포, 이상치 등을 파악했습니다.

### 4-2. 공정 단계별 모델링 및 성능 평가

*   화성 공정을 7개의 주요 단계(C1, DC1, C2, DC2, C3, DC3, C4)로 나누어 개별 분석을 진행했습니다.
*   각 단계마다 5가지 머신러닝 알고리즘(DecisionTree, RandomForest, XGBoost, LightGBM, CatBoost)을 적용하여 불량 예측 모델을 학습시켰습니다.
*   불량 셀을 놓치지 않는 것(Minimize False Negatives)이 중요하다고 판단, 재현율(Recall)을 핵심 평가지표로 삼아 각 단계별 최적 모델을 선정했습니다. (대부분 CatBoost가 선정됨)


## 5. 시뮬레이션을 통한 공정 개선 효과 검증

프로젝트의 핵심 목표인 '공정 최적화'를 검증하기 위해 다음과 같은 시뮬레이션을 진행했습니다.

### 5-1. 시뮬레이션 설계

*   제어 변수: 이전 분석에서 불량률에 가장 큰 영향을 미치는 것으로 나타난 각 공정 단계의 '평균 온도' 변수를 제어 대상으로 설정했습니다.
*   시나리오:
    1.  AS-IS: 현재 공정 조건에서 예측되는 불량률
    2.  TO-BE: '평균 온도'를 사전에 정의된 최적 값으로 강제 제어했을 때 예측되는 불량률

### 5-2. 시뮬레이션 결과

*   7개 모든 공정의 온도를 동시에 최적 값으로 제어하는 '통합 시뮬레이션' 결과, 전체 불량률이 3.13%에서 2.04%로 감소하여 약 1.08%p의 개선 효과를 얻을 수 있음을 확인했습니다.
*   이는 PDF 발표자료에서 제시한 연간 162억 원의 품질 기회비용 절감으로 이어질 수 있는 정량적 근거가 됩니다.


## 6. 결론 (Conclusion)

본 프로젝트를 통해 2차 전지 화성 공정의 복잡한 데이터 속에서 온도가 불량률에 미치는 핵심적인 영향도를 통계적으로 증명하고, CatBoost 기반의 예측 모델을 구축했습니다. 더 나아가, 개발된 모델을 활용한 시뮬레이션을 통해 공정 최적화 시 얻을 수 있는 개선 효과를 1.08%p라는 구체적인 수치로 제시하며 데이터 분석의 실질적인 가치를 입증했습니다.

## 7. 프로젝트를 통해 배운 점 및 제언 (Learnings & Suggestions)

실제 제조 공정 데이터를 다루며 도메인 지식의 중요성과 데이터 정제의 어려움을 깊이 체감했습니다. 특히, 각 공정 단계별로 최적의 모델을 선정하고 이를 통합하여 전체 개선 효과를 시뮬레이션하는 과정은 복잡한 비즈니스 문제를 구조적으로 분해하고 해결하는 능력을 길러주었습니다.

이번 분석 경험을 바탕으로, 단순 예측을 넘어 실시간으로 공정 데이터를 모니터링하고 이상 징후 발생 시 자동으로 최적의 제어 값을 추천하는 'AI 기반 공정 최적화 에이전트(Agent)'를 개발하는 방향으로 확장할 수 있는 가능성을 보았습니다.
